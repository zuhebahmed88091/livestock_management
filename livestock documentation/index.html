How To Setup

  • Step 1: Extract livestock.zip.
  • Step 2: Go to files directory.
  • Step 3: zip all files and upload this on your server.
















  • Step 4: Create mySQL Database and Database User.










  • Step 5: Open your browser and visit http://your-sitename/install.Click Check Requirements button


  • Step 5.1:
    Click Check Permissions Button If Server Requirements are everything fine


  • Step 5.2:
    Click Configure Environment Button If Permissions are fine


  • Step 5.3:
    Click Form Wizard Setup


  • Step 5.4:
    Type the App Name that you want to give to your website (example: My App, My website etc) and App Url of your website. Then Click Setup Database


  • Step 5.5:
    Fill up the Database informations with the informations that you have created in step 4. Click Setup Application


  • Step 5.6:
    Fill up the Mail informations. This will work as notifier mail for the website and will send users email notifications. You can Leave the fields as it is if you dont want to setup Mail . Click Install


  • Step 5.7:
    With Successfull Installation of the website you will see a page like this and you are done!
    Congratulations! you have successfully installed the website with your own credentials.
    Try Click here to exit button to exit the setup wizard and go to your newly setup website.
